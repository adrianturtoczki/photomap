# photomap

Setup:
1, Install mysql
2, Create a .env file in the root folder with the following filled out:\
DB_HOST=\
DB_USER=\
DB_PASS=\
DB_DATABASE_NAME=\
EXPRESS_PORT=\
EXPRESS_SECRET=\
3, To create the databases, run 'node db/createdb.js' file after npm install. After that, you can run the app with "npm start"
